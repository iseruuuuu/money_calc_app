# What is This?

### 稼いだ額を入力することで、稼いだ合計額と103万円までいくら稼げるのかを記録できるアプリ  

# What you can do
#### ・給料の金額を簡単に登録することができます！
#### ・給料を登録すると、１０３万円まであといくらなのかを確認することができます！
#### ・記録を忘れないように、給料日に通知が届きます！

# Reason which I make it

### 私の知り合いが、「103万円まであといくらまで稼げるのかをわざわざ計算するのがめんどくさい」という相談を受けて作成しました。
